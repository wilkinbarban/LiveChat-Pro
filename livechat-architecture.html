Funcionalidades implementadas
🔗
Sesión persistente
sessionId guardado en cookie + localStorage. El historial se recupera al recargar la página.
🌍
Traducción automática
Mensajes del usuario → español para el admin. Respuestas del admin → idioma original del usuario.
✍️
Ghost Typing
El admin ve en Telegram lo que el usuario está escribiendo en tiempo real, antes de que presione Enviar.
📍
Geolocalización
Al conectar, se envía ficha técnica: ciudad, país, ISP, User-Agent y página actual via ip-api.com.
🔬
Análisis de sentimiento
Librería sentiment detecta mensajes ofensivos y clientes molestos. Etiquetas automáticas en Telegram.
🛡️
Mapeo multihilo
sessionId ↔ socketId mapeados. Las respuestas del admin llegan solo al usuario correcto. Sin mezclas.
👋
Flujo de bienvenida
El bot pregunta el nombre del usuario. Una vez recibido, actualiza el header del chat y el panel en Telegram.
🎨
Widget configurable
Color, estilo de botón (flotante/persistente/oculto) y mensaje de bienvenida configurables via CLI setup.
Comandos del bot de Telegram
/usuarios
Lista todos los usuarios activos con nombre, ID y ubicación.
/ban [id]
Bloquea un usuario por sessionId. Desconecta su socket inmediatamente.
/info [id]
IP, geolocalización, ISP, User-Agent, página actual y prioridad.
/clean
Elimina sesiones inactivas hace más de 1 hora de la memoria.
Flujo de un mensaje entrante
1
Usuario escribe → socket.emit('message')
El widget captura el texto y lo emite al servidor via WebSocket.
2
Detección de idioma
Google Translate API (endpoint libre) detecta el idioma del mensaje automáticamente.
3
Traducción al español
Si el idioma no es español, se traduce para que el administrador reciba todo en su idioma.
4
Análisis de sentimiento
Se evalúan palabras ofensivas y puntuación negativa. Se añaden etiquetas [OFENSIVO] o [PRIORIDAD ALTA].
5
Envío a Telegram
El mensaje llega al admin con nombre, sessionId, contexto y etiquetas de alerta si aplica.
6
Admin responde en Telegram
El servidor detecta la respuesta, traduce de vuelta al idioma original del usuario y la enruta via socket al cliente correcto.

🚀 Instalación y puesta en marcha

# 1. Instalar dependencias
npm install

# 2. Ejecutar asistente de configuración
node setup.js
→ Token del Bot, Admin ID, color, estilo de botón

# 3. Iniciar servidor
node server.js

# 4. Integrar en cualquier web
<script src="https://tu-servidor/widget.js"
        data-server="https://tu-servidor"></script>
